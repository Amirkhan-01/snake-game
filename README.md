# Snake Game - Premium Edition

Премиальная версия классической игры "Змейка" на Python (Pygame): скины, комбо, достижения, ежедневные задания, процедурный звук, эффекты освещения и bloom.

> **О распространении:** это десктопная игра на Pygame. Её нельзя напрямую загрузить в Apple App Store (см. раздел «Сборка и распространение» ниже) — но её можно собрать в самостоятельное приложение для Windows/macOS/Linux или в веб-версию.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск игры

```bash
python snake.py
```

## Управление

- **Стрелки** - управление змейкой
- **Свайп (touch/swipe)** - управление на мобильных устройствах
- **P** - пауза/продолжение игры
- **S** - настройки
- **B** - магазин скинов
- **SPACE** - перезапуск после проигрыша
- **ESC** - выход из игры

## Правила

- Собирайте еду для роста змейки
- Не врезайтесь в стены, препятствия и себя
- Разные типы еды дают разное количество очков
- Комбо-система: быстрое поедание еды увеличивает множитель очков
- Выполняйте ежедневные задания для получения валюты
- Покупайте и используйте скины в магазине

## Типы еды

- **Красная (обычная)** - 10 очков
- **Фиолетовая (бонус)** - 30 очков, исчезает со временем
- **Голубая (замедление)** - 15 очков + временное замедление игры (slow motion)

## Особенности

### Базовые функции
- **Система рекордов** - лучший счет сохраняется в файле
- **Уровни сложности** - скорость увеличивается с каждыми 50 очками
- **Пауза** - можно приостановить игру по нажатию P

### Продвинутые функции
- **Разные типы еды** - обычная, бонусная и замедляющая
- **Препятствия** - серые блоки на поле, количество растет с уровнем
- **Частицы** - визуальные эффекты при поедании еды
- **Улучшенная графика** - градиент на змейке, глаза на голове, пульсирующие эффекты
- **Комбо-система** - множитель очков за быстрое поедание еды
- **Slow motion** - голубая еда временно замедляет игру
- **Тряска экрана** - визуальный эффект при смерти
- **Динамические препятствия** - генерируются случайно, избегают стартовую позицию

### Мобильные функции
- **Touch/swipe управление** - поддержка свайпов для мобильных устройств
- **Адаптивный дизайн** - автоматическая подстройка под размер экрана
- **Отзывчивый интерфейс** - элементы UI масштабируются под экран

### Система достижений
- **First Blood** - наберите 10 очков
- **Combo Master** - достигните 5x комбо
- **Rising Star** - достигните 5 уровня
- **Century** - наберите 100 очков
- **Survivor** - играйте 2 минуты

### Ежедневные задания
- **Ежедневные челленджи** - новые задания каждый день
- **Награды** - получение валюты за выполнение заданий
- **Разнообразие** - задания на очки, комбо, уровень и выживание

### Магазин скинов
- **Classic** - зеленая змейка (бесплатно)
- **Fire** - оранжевая змейка (100 валюты)
- **Ice** - голубая змейка (150 валюты)
- **Gold** - золотая змейка (200 валюты)
- **Neon** - неоновая змейка (300 валюты)
- **Rainbow** - радужная змейка (500 валюты)

### Настройки
- **Звук** - включение/выключение звука
- **Вибрация** - включение/выключение вибрации

### Обучение
- **Учебное пособие** - подсказки для новых игроков
- **Автоматическое скрытие** - исчезает после первой игры

## Файлы

- `snake.py` - основной файл игры
- `requirements.txt` - зависимости (`pygame`, `numpy` для звука)
- `high_score.json` - сохраненный рекорд (создается автоматически)
- `achievements.json` - достижения (создается автоматически)
- `daily_challenge.json` - ежедневные задания (создаются автоматически)
- `currency.json` - валюта (создается автоматически)
- `skins.json` - скины (создаются автоматически)
- `settings.json` - настройки звука/вибрации (создается автоматически)
- `dist/SnakeGame.exe` - готовый самостоятельный .exe (Python не нужен) ✅
- `run_game.bat` - запуск десктопной версии из исходников двойным кликом
- `web/index.html` - веб-версия на JavaScript/Canvas (браузер, телефон) ✅
- `web/serve.py`, `web/play_web.bat` - локальный запуск веб-версии

## Звук

Звуковые эффекты (еда, бонус, замедление, покупка, достижение, проигрыш) генерируются
процедурно через `numpy` — внешних аудиофайлов не требуется. Если `numpy` или аудиоустройство
недоступны, игра работает без звука. Звук выключается в настройках (клавиша **S**, затем **1**).

## Запуск (быстро)

- **Готовый .exe (рекомендуется):** двойной клик по `dist/SnakeGame.exe` — Python не нужен.
- **Из исходников:** двойной клик по `run_game.bat` (или `python snake.py`).
- **Веб-версия (браузер/телефон):** `web/play_web.bat` или открой `web/index.html`.

## Десктопная сборка (.exe) — основной, проверенный вариант

Самостоятельный `SnakeGame.exe` уже собран в папке `dist/` и запускается без Python.
Пересобрать (например под другую версию):
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name SnakeGame snake.py
```
Windows `.exe` собирается только на Windows, macOS `.app` — только на macOS.
Распространять можно напрямую или через [itch.io](https://itch.io) (desktop-игра).

## Веб-версия (браузер, телефон)

В папке `web/` лежит **отдельная реализация игры на чистом HTML5 + JavaScript/Canvas**
(`web/index.html`) — без Python, без wasm, без сборки. Работает в любом современном
браузере на ПК и телефоне.

Запуск локально:
```
web\play_web.bat        (запускает статический сервер и открывает браузер)
```
или просто открой `web/index.html` двойным кликом. Для игры с телефона запусти
`web/serve.py` и открой на телефоне `http://<IP-компьютера>:8000/` (та же Wi-Fi).

Возможности веб-версии: **4 режима** (Classic / Obstacles / Hardcore / ⏱ **Time Attack**),
**усилители** (🛡 щит, 2× очки, 🧲 магнит) с таймерами, **темы оформления**, скины и магазин,
комбо, уровни, ежедневные задания, достижения, **лидерборд (топ-10)**, статистика,
частицы, звук + фоновая музыка (WebAudio).
Управление: **свайпы, стрелки и WASD** (при любой раскладке — привязка к физическим клавишам).
Прогресс — в `localStorage`.

Готово к публикации: **политика конфиденциальности** (`privacy.html`), экран **About**,
Open Graph / Twitter превью (`og-image.png`), иконки и favicon.

**Монетизация** (слой покупок отделён — в вебе демо, готов под нативный биллинг):
- rewarded-реклама («продолжить» / «×2 монеты»),
- **Store**: Full Version (полная версия), Remove Ads, паки монет (1000 / 6000),
  скины за монеты + премиум-скины за деньги (Galaxy, Aurora).
- Точки интеграции помечены в коде: `playAd()` (реклама) и `IAP.purchase()` (покупки).

План публикации и полный список пробелов — в [ROADMAP.md](ROADMAP.md) и [PUBLISHING.md](PUBLISHING.md);
готовые скриншоты — в `store-screenshots/`.

**Установка как приложение (PWA):** игра — устанавливаемое PWA. На телефоне в браузере
выбери «Добавить на главный экран» (Android — кнопка «📲 Install app» в меню; iOS Safari —
Поделиться → «На экран Домой»). Тогда она открывается со своей иконкой, в полный экран,
и **работает офлайн** (service worker кэширует ассеты).

Ещё добавлено: 🎁 ежедневная награда со streak-стриком, 📊 экран статистики и достижений,
📤 «поделиться счётом», 🪙 монеты за каждую игру (Hardcore ×1.5), вибрация (haptics),
авто-пауза при сворачивании.

Файлы PWA: `manifest.webmanifest`, `sw.js`, `icon-*.png`, `apple-touch-icon.png`.

**Публикация:** содержимое `web/` (`index.html` + манифест/иконки/sw.js) — статический сайт.
Заливается как есть на [itch.io](https://itch.io) (HTML-игра), GitHub Pages, Netlify,
Vercel или любой хостинг. Открывается и с мобильного браузера — это самый близкий к
«App Store» вариант охвата на телефонах.

> Веб-версия — самостоятельный порт; десктопная версия на Pygame (`snake.py`) остаётся
> основной и сохраняет прогресс в JSON-файлы.

## Apple App Store

Напрямую **невозможно**: Pygame не собирается под iOS, а сборка `.ipa` требует
macOS + Xcode + Apple Developer Program ($99/год). Для настоящего iOS-приложения игру
пришлось бы переписать (например на Kivy). Наиболее реалистичные пути публикации —
веб-версия (itch.io / свой сайт, играется и с телефона) или десктоп (itch.io / Steam).
